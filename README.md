# Energy Sector Commercial & Risk Analytics Dashboard

## 📊 Project Overview
This project focuses on analyzing business performance and optimizing cost and risk in the energy sector using Power BI.

## 🔹 Dashboards Created
1. Business Performance Dashboard
   - Revenue, Profit, Customer Analysis
2. Strategic Cost & Risk Dashboard
   - Profit Gap, Churn Risk, Renewable ROI

## 🔹 Key KPIs
- Total Revenue: $567M
- Total Profit: $184M
- Profit Margin: 32%
- High Risk Revenue: $85M

## 🔹 Tools Used
- Power BI
- DAX
- Excel

## 🔹 Key Insights
- Identified low-margin regions for cost optimization
- Detected high-risk customers using churn analysis
- Highlighted profitable energy sources for strategic sourcing
